Instructions for Uploading Files

Please make sure your Excel file contains the following columns:

  • Patent / Publication Number: The unique identifier for the patent.
  • Country Code: The two-letter country code where the patent was filed.
  • Priority Date: The earliest date of filing.
  • Filing Date: The date when the patent was filed.
  • Issued Date: The date when the patent was granted.
  • Expiration Date: The date when the patent expires.
  • Number of Claims: The number of claims made in the patent.

The file must be in .xlsx format and should not contain any empty rows or irrelevant data.
